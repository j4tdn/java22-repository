1. tính chất đóng gói (encapsulation) 
tính chất dóng ghói được thể hiện qua các access modifier ( các từ khóa khai báo phạm vi truy cập)
 
 
  khai báo access modifiler ở thuộc tính và phương thức
 nếu access modifile là:
 
 + private đóng gói trong phạm vi của class 
 ---> attribute( thuộc tính ) , method là private thì chỉ được phép sử dụng trong class chứ nó 
 
 + protected / no-declare : đóng gói trong phạm vi packege 
 
 -----> attribute method là protected thì được dùng trong class hiện tại và class khác nhyuwng cùng package 
 protected / no-declare phân biệt 2 cái 
 
 - protected :
 	attribute method là protected còn có thể được thừa kế, sử dụng ở các class con của class hiẹne tại class con  khác nằm ngoài package hiện tại 
 	
 - no-declare : dùng trong package ko thể dùng bên ngoài 
 + public : dungf được mọi nơi trong dự án
