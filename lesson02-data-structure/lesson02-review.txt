2. Kiểu dữ liệu
- Nguyên thủy(primitive data type)
  VD: int, double, string
  + Giá trị: lưu trữ ở STACK
  
- Đối tượng(object type)
  VD1: Integer, Double, String: KDL có sắn của Java --> 100% nó là class có duy nhất 1 thuộc tính bên trong
       --> Immutable Object
       
  VD2: Item, Store, Employee: KDL do mình tự tạo ra phù hợp với bài toán
       --> Mutable hoặc Immutable do bài toán
  
  + Giá trị: lưu trữ ở HEAP
  
- Biến
  + Luôn được lưu trữ ở STACK
  
3. Từ khóa final
Là final ở STACK, ko thể cập nhật giá trị của biến ở STACK
